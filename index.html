Restaurant Ranking using Fuzzy Logic

The principle of partial truth was introduced in fuzzy logic, which enhances boolean logic. The truth level replaces the boolean truth in fuzzy logic. Because the modern and methodical science of fuzzy logic was only discovered a few years ago, fuzzy logic is said to be an old new logic, even though in reality the idea of ​​fuzzy logic has been around for a long time. The purpose of project is to understand Fuzzy Logic and its application, understand the linguistic name of Fuzzy Logic, understand the membership function of Fuzzy Logic, understand the stages in Fuzzy Logic.

Map Coloring using Greedy and Backtracking strategies

Graph coloring to color the map of West Bandung Regency, West Java. West Bandung Regency has 16 sub-districts so that it will be colored with a graph coloring program using a greedy and backtracking strategy. The results of testing the program are expected to produce map coloring instructions with the fewest colors and do not violate the constraints of the graph coloring method. The map that has been obtained will be represented as a graph by placing a vertex in the middle of each sub-district on the West Bandung Regency map. Then, create edges between the visible vertices sharing the border. Thus, the results of the representation can be computed to solve the graph coloring problem.

  • Created - 10 June 2021
  • Technologies used - Java
  • Role - Programmer
  • View online - https://github.com/wydaun/graph-coloring
